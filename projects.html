My Projects

Throughout my time at NSCC, I have developed programming skills in a variety of different languages such as C#, Java,Python, and more. Below is a compilation of some of my most favourite projects I have worked on. Click a tile to view each project.
Humans Versus Zombies LifeSim

The Humans versus Zombies Life Simulator is a project developed in C++ as part of a programming class. The simulator aims to create a virtual environment where humans and zombies coexist, simulating their interactions and the struggle for survival, allowing users to observer the population fluctuations over time
WPF Media Player

The WPF Media Player is a project developed in C# with Windows Presentation Foundation (WPF). The media player allows users to load up their song in MP3 formar, then play, pause, or stop the MP3. Users can also edit the MP3's metadata directly in the media player.
Transit Tracking App

The Transit Tracking app is an app developed in Kotlin. The transit app uses API data to get current accurate bus locations across HRM. The appm also features notifications from Halifax Transit, as well as a list where users can add their favourite routes.
